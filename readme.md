## Installing

```shell
npm install -g sms-cli
```


## Usage

```shell
sms
```

When you run `sms` for the first time, it will ask you for your Twilio credentials. You can also provide them via `TWILIO_SID` and `TWILIO_TOKEN` environment variables.
